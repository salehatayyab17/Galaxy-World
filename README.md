#  Galaxy World — Samsung Notebook Landing Page

A dark-themed, responsive product landing page inspired by Samsung Notebook laptops, built with HTML, CSS, and JavaScript.

##  Built With
- HTML5
- CSS3 (animations, responsive design, media queries)
- JavaScript (DOM manipulation, event handling)

##  Features
- Animated hero section with glowing title
- Auto-scrolling product feature slider
- Interactive color selector (Eclipse Grey / Platinum White)
- Functional notification popup with close button
- Scroll-to-top button with smooth scroll behavior
- Fully responsive layout (mobile, tablet, desktop)
- Custom dark-theme SVG icons and illustrations
